# **App Name**: Wazafni (وظفني)

## Core Features:

- AI Job Match Compatibility: An AI-powered matching tool that calculates a percentage score between a candidate's resume and job requirements to guide hiring decisions.
- RTL-Optimized Job Board: A high-end, responsive search experience built specifically for Arabic typography and Right-to-Left (RTL) layout standards.
- Intelligent Resume Analyzer: An AI tool for job seekers to extract skills, improve ATS readability, and provide career path recommendations.
- Candidate Tracking System (ATS): A robust backend management system to store and track applications, status changes, and interviewer feedback in a database.
- Dynamic Analytics Dashboard: A modern visualization layer for employers to track hiring success rates and job performance via interactive charts.
- Unified Profile Management: Centralized database storage for job seeker profiles, resumes, and employer credentials for a seamless sign-on experience.
- Smart Messaging Center: An elegant communication UI for real-time interaction between employers and candidates with integrated notification states.

## Style Guidelines:

- Primary Color: Deep Professional Purple (#5929BA) representing intelligence and institutional stability.
- Background: Ultra-light lavender tint (#F8F7FA) for a clean, premium Apple-style SaaS aesthetic.
- Accent Color: Vibrant Soft Violet (#4D7BFF) for highlights and primary CTAs to drive engagement.
- Recommendation: Use 'Cairo' for large, bold Arabic headlines and 'IBM Plex Sans Arabic' for clear, objective body text. Note: currently only Google Fonts are supported.
- Custom dual-tone icons with rounded strokes, reflecting a high-tech yet human-centric recruitment vibe.
- A spacious, Linear-inspired layout featuring floating cards with glassmorphism effects and ample negative space.
- Subtle, smooth micro-interactions on hover and soft fade-in transitions for dashboard metrics.
